## ✨ 游戏特性
- **自由控制飞船**：在太空中自由穿梭，抵御外星入侵者。
- **动态外星人舰队**：敌人会不断移动和逼近，挑战逐步升级。
- **得分与等级系统**：记录最高分，随着游戏深入难度不断递增。
- **游戏重置机制**：包含直观的“Play”按钮，轻松重新开始。

## 🛠️ 环境要求
- Python 3.x
- Pygame (2.6.1 或更高版本)

## 📦 安装说明

1. **克隆此仓库**：
   ```bash
   git clone git@github.com:hulk-2019/alien-invasion.git
   cd alien-invasion
   ```

2. **安装依赖项**：
   ```bash
   pip install -r requirements.txt
   ```

## 🎮 怎么玩

运行主程序来启动游戏：
```bash
python main.py
```

### 游戏操作
| 按键 | 动作 |
| --- | --- |
| `⬅️ 左方向键` | 向左移动飞船 |
| `➡️ 右方向键` | 向右移动飞船 |
| `空格键` | 发射子弹 |
| `Q` 键 | 退出游戏 |
| `鼠标点击` | 点击 **Play** 按钮开始游戏 |
